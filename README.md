# 🀄 麻将惩罚抽取程序

四个人依次私下输入惩罚，打乱后随机抽取的互动程序。

## 功能说明

### 1. 输入阶段（私密）
- **同学A** 先输入自己的惩罚列表，可增删改查
- 确认后，列表被保存并隐藏，轮到 **同学B**
- 依次轮到 **同学C**、**同学D**
- 每人输入时只能看到自己的惩罚，他人不可见

### 2. 抽取阶段
- 四人全部确认后，所有惩罚会打乱顺序
- 选择自己的身份（同学A/B/C/D）
- 点击「抽取惩罚」获得一个随机惩罚
- 点击「已知晓」关闭当前显示
- 若忘记自己的惩罚，可输入名字点击「查看我的惩罚」

### 3. 重新开始
- 点击「重新开始游戏」清空所有数据，从头再来

## 安装与运行

```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务
python app.py
```

浏览器访问：http://localhost:5000

## 技术栈

- 后端：Python + Flask
- 前端：原生 HTML + CSS + JavaScript
- 数据：本地 JSON 文件存储（`game_data.json`）
